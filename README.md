# SimpleDash-Jump
## A Simple Dash & Jump mod for Doom
## What does it Do?
- Simple Dashing in 4 directions
- Simple Double Jump

## Cvars
shot_allowdoublejump = true/false
shot_allowdash = true/false

## Compatibility
Any mod

Runs with GZDoom / Zandronum
